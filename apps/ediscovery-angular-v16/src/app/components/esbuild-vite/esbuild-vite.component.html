# Esbuild & Vite

In modern frontend development, build tools play an indispensable role in creating efficient and fast web applications. Esbuild and Vite are two such emerging tools that have caught developers attention due to their speed and simplicity.

Esbuild provides native support for CSS and TypeScript without the need for plugins. Vite, on the other hand, handles CSS and TypeScript through its plugin ecosystem

Vite pre-bundles dependencies using esbuild. Esbuild is written in Go and pre-bundles dependencies 10-100x faster than JavaScript-based bundlers.
